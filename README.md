# TECH NEWS

## Description
A social application for tech news created using Java and Spring Boot.

Users can:
- Create an account
- Login/logout from application
- Create tech news posts
- View tech news posts
- Comment on posts
- Upvote posts
- Edit their posts
- Edit their comments on posts
- Delete their posts
